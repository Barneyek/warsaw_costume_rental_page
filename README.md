# Warsaw Costume Rental

Educational project — decoupled web application for costume rental.

## Stack
- **Backend:** Django + Django REST Framework
- **Frontend:** React + Vite + TypeScript
- **Architecture:** Headless (API-first)

## Status
🚧 Work in progress — learning project.

## Structure
warsaw_costume_rental/
├── backend/          # Django + DRF
│   ├── src/          # Django apps (core, catalogue, blog, pages, inquiry)
│   └── web_app/      # Settings, urls, wsgi
├── frontend/         # React + Vite + TS
└── docs/             # Documentation & implementation plans
└── plans/        # Per-issue implementation plans
